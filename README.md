# 🧠 Mini-CRM for Freelancers

A lightweight fullstack CRM (Customer Relationship Management) system designed for freelancers to manage clients, projects, interactions, and reminders in a structured and secure way.

This project simulates a real-world SaaS application with multi-tenant architecture, secure authentication, and relational database design.

---

## 🚀 Features

- 🔐 Secure authentication (signup & login)
- 👥 Client management (CRUD)
- 📁 Project management with status tracking
- 📝 Interaction logs (calls, meetings, emails)
- ⏰ Reminders with due-date tracking
- 📊 Dashboard with business overview
- 🌗 Light/Dark mode
- 📱 Fully responsive UI

---

## 🛠 Tech Stack

**Frontend**

- Nextjs
- TypeScript
- TailwindCSS

**Backend**

- Node.js (Express / NestJS)
- TypeScript
- REST API architecture

**Database**

- PostgreSQL
- Prisma ORM

---

## 🏗 Architecture Highlights

- Multi-user data isolation
- Modular backend structure (controllers, services, models)
- Secure password hashing
- Protected routes with authentication middleware
- Relational database schema with proper indexing

---

## 🎯 Project Goal

The goal of this project is to demonstrate:

- Fullstack TypeScript development
- Real-world database design
- Secure authentication implementation
- Clean, scalable system architecture
